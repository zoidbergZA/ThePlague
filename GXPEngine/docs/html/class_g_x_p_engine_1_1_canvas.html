Detailed Description

The Canvas object can be used for drawing 2D visuals at runtime.

Constructor & Destructor Documentation

GXPEngine.Canvas.Canvas ( int  width,
int  height 
)

Initializes a new instance of the GXPEngine.Canvas class. It is a regular GameObject that can be added to any displaylist via commands such as AddChild. It contains a System.Drawing.Graphics component.

float GXPEngine.Canvas.alpha
getset

Gets or sets the alpha value of the canvas. Setting this value allows you to make the canvas (semi-)transparent. The alpha value should be in the range 0...1, where 0 is fully transparent and 1 is fully opaque.

uint GXPEngine.Canvas.color

Gets or sets the color filter for this canvas. This can be any value between 0x000000 and 0xFFFFFF.
