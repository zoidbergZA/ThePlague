static bool GXPEngine.Input.GetKeyDown ( int  key)
static

Returns 'true' if specified key was pressed since once or more the previous call to 'GetKeyDown' for this key. Internally, the number of times the key is pressed is counted. Using this command reset this counter. So for this command to work properly, you need to read it every frame.

keyKey number, use Key.KEYNAME or integer value.
